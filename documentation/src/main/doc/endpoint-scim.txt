=== SCIM RESTful API endpoint

Endpoint type:: +SCIM+
Binding:: RESTful


This endpoint exposes SCIM RESTful API. SCIM is a standard protocol, more details can be found on http://www.simplecloud.info and in RFC: https://datatracker.ietf.org/doc/html/rfc7644.
Unity exposes only a read only part of the SCIM API.
